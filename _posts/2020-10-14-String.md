### Append()
```c++
std::string s="C+";
string ch="+";

//append string
s.append(ch);
//append char, append(number of characters to copy, character value)
char sh='+';
s.append(1,sh);

//append partial string, append(string, [optional]subpos, sublen)
string str="writing";
s.append(str,6,3);
```


### append char to end of string
``` c++
std::string s="C+";
char ch='+';

//push_back()
s.push_back(ch);
//+=operator
s+=ch;
//append()
s.append(1,ch);
```
